# Get Riot API with three.js

Riot API를 활용해서, 롤 전적 데이터 3D 시각화

## Stack

- 코어 : `CRA` `TypesSript`
- 스타일링 : `SCSS` `CSS Module` `React-three-fiber`
- 상태관리 : `React-Query` `Recoil`
- CI/CD : `Vercel Deploy`
- 브랜치 전략 : `Git Flow`
